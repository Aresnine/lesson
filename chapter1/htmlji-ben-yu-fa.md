# HTML的基本语法与常用标签

---

```
<marquee></marquee> 标签：让被标记的内容滚动起来
```

### **标签：**

```
1、分类
    双标签：
        <标签名>被标记的内容</标签名>
        注意：
            1、<标签名>代表标签的开始
            2、</标签名>代表标签的结束
            3、标签名不区分大小写
            小技巧：双标签先把成对的标签写出来，在写被标记的内容
    单标签：
        <标签名 /><br />
```

### **标签属性:**

```
loop="2" bgcolor="yellow" 他们叫做标签的属性

属性：
    1、属性名
    2、属性值
    格式：
        属性名=属性值
        
    特点：
        1、格式：属性名="属性值" 属性名='属性值' 属性名=属性值均正确，因为html是宽松的语言，但是建议使用单引号或者双引号
        2、一个标签内可以有多个属性
        
    
```
