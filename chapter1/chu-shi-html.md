## 1、软件结构的划分

微信、qq、yy、陌陌、内涵段子

```
c/s结构和b/s软件：

1）、c/s软件结构：
    c:client 客户端
    s:server 服务器
    
    特点：
        1、都有特定的客户端
        2、客户端和服务器都能存储数据
        3、可以通过任意的协议进行通信
        
2）、b/s软件结构
    b:browser 浏览器
    s:server  服务器
    
    特点：
        1、所有的b/s结构软件的客户端都是浏览器
        2、所有的数据都存储在服务器，相对安全
        
b/s结构软件和c/s软件结构的优缺点：
    1、不需要开发特定的客户端（省钱）
    2、所有数据都存储在服务器，服务器压力大,需要花费优化服务器
```

## 2、web发展过程

```
    1、web1.0阶段（静态网站）
         代表：新闻网站、政府门户网站
         特点：
            类似于txt文档提供简单的阅读、下载---》网民不能参与其中
            网站----》人
            网站是信息的提供者，单向性的提供和单一性的理解
            
            
    2、web2.0阶段（动态网站----我们正处于的阶段）
         代表：商城、论坛、贴吧
         特点：
            以网络为沟通渠道进行人与人沟通
            人《----》人
            网络是一个平台，用户提供信息，通过网络，其他用户获得。
            
   3、web3.0阶段（未来阶段）
         特点：
            人工智能、关联数据和语义网络构建，形成人和网络以及网络与人的沟通，同时在seo支持下，提高人与人沟通的便利性
            人《----》网络《-----》人
            web3.0网络成为用户需求理解者和提供者，网络对用户了如指掌，知道用户有什么、要什么以及行为习惯，进行资源筛选、智能匹配，直接给用户答案。            
```
